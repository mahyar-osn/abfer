# abfer

A simple module to read, process, plot, and save Axon Binary Format (.abf) files.

- Dependencies: 
    >> pyabf
    
    >> matplotlib
    
    >> numpy
    
- Usage:

    ```python
    python main.py <input_abf_file_path>
    ```
    
    Other optional arguments:
    
    ```python
    plot
    save
    ```
    
    To plot the data or save them into a CSV file.
